Documentation for API Endpoints

All URIs are relative to https://mil-d-apim.azure-api.net/mil-auth/v2

Class Method HTTP request Description
AllApi getAccessToken POST /token Returns access token or refreshes it
DeviceApi activate POST /activate Activate a device
DeviceApi getDeviceCode POST /device/code Returns user code and device code for device authentication flow
InfoApi getJwks GET /.well-known/jwks.json Returns keys
InfoApi getOpenIdConf GET /.well-known/openid-configuration Returns OpenID configuration
InfoApi introspect POST /token_info Introspects a given access token returning the clear value of user tax code (if any)
